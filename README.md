# GradingProgram

<h2>About</h2>
A grading program written in Java using an mySQL database.
Created for Dave Goldsmith.
Created by Austin Patel and Jason Morris on October 13, 2016.
Contributed to by Achinthya Poduval and Lex von Klark.

<h2>Features</h2>
<ul>
  <li>Automatic table setup</li>
  <li>Add students</li>
  <li>Create assignments</li>
  <li>Create classes</li>
  <li>Enroll students</li>
  <li>Grade book</li>
  <li>Grade scales</li>
  <li>View database tables</li>
  <li>Console</li>
</ul>
